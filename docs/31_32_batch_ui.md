# 31→32 一括実行 UI

## 使い方
1. `bat/31_32_crossroad_performance_to_report.bat` を起動
2. **① プロジェクト選択** で対象プロジェクトフォルダを選択
3. **② 曜日選択** で `月 火 水 木 金 土 日 ALL` のチェックボックスを設定（初期値は ALL=ON）
4. **31→32 一括実行** を押下


### 曜日チェックボックスの連動
- **ALL をON**にすると、`月〜日` がすべて自動でONになります。
- **ALL をOFF**にすると、`月〜日` がすべて自動でOFFになります。
- `月〜日` が**すべてON**になると、ALL も自動でONになります。
- `月〜日` のどれかが**OFF**になると、ALL は自動でOFFになります。

実行時の31への引数は次の通りです。
- ALL がON: `--weekdays ALL`
- ALL がOFF: チェックされた曜日（漢字）を複数指定（例: `--weekdays 火 水 木`）

## 実行対象の判定（デフォルトチェックON条件）
下記4点がすべて揃う交差点のみ、初期状態でチェックONになります。

- `11_交差点(Point)データ/<交差点名>.csv`
- `11_交差点(Point)データ/<交差点名>.jpg`
- `20_第２スクリーニング/<交差点名>/` フォルダ
- `20_第２スクリーニング/<交差点名>/` 配下に `*.csv` が1つ以上

※不足があっても、表のチェックは手動でON/OFFできます。

## 出力先（固定）
- `31_交差点パフォーマンス/<交差点名>_performance.csv`
- `32_交差点レポート/<交差点名>_report.xlsx`

`31_交差点パフォーマンス` と `32_交差点レポート` は、起動時スキャン時に無ければ自動作成されます。
