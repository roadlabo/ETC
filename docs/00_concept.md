# docs/00_concept.md — ETC2.0分析ツール群の設計思想

本リポジトリは、ETC2.0プローブデータを用いて
交通の「量」ではなく「挙動・構造」を把握するための分析ツール群である。

単発の集計ではなく、
**番号順に実行する分析パイプライン**として設計している。

---

## 番号付きスクリプトの意味

src ディレクトリ内のスクリプトは、番号順に実行することを前提としている。

01_* 前処理
10_* 空間定義・サンプリング
20_* Trip抽出
30_* 性能評価・可視化
40_* OD分析
70_* Path・流入流出分析

---

## 本ツールの立ち位置

- シミュレーションではない
- 個々の車両行動を断定しない
- EBPMの判断材料を提供する

分析結果はすべて **推定・相対評価**である。
