# 03_base_zone_estimator

第1スクリーニング済みフォルダ内の運行ID別CSVを走査し、夜間停留（22:00〜05:00）を使って運行IDごとの推定拠点ゾーンを作成します。

## 入力
- `--input`: 第1スクリーニングフォルダ
- `--zoning`: 任意ゾーニングCSV（ポリゴン定義）
- `--output`: 出力CSV（省略時は `<inputフォルダ名>_拠点ゾーン.csv` を同階層に作成）
- `--recursive`: サブフォルダを含む

## 出力
- `op_id`
- `base_zone`

## 判定概要
1. 夜間帯レコード抽出（22:00〜05:00）
2. 連続点間距離100m以下が30分以上継続する区間を停留候補とする
3. 停留区間代表点（平均座標）を優先してゾーン割当
4. 代表点未判定時は停留直前点・直後点で補完
5. 運行ID内で最長停留候補を優先し `base_zone` を決定
   - 候補あり未割当: `ゾーン外`
   - 候補なし: `判定不可`
